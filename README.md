# Altera-Climat · Прототип редизайна

Готовый продающий лендинг под кондиционерный бизнес в Оренбурге. Чистый HTML/CSS/JS, без сборки и зависимостей.

## Что внутри

- **index.html** — разметка
- **styles.css** — стили
- **script.js** — интерактив (квиз, калькулятор, фильтры, формы)

## Что показать клиенту (главные «вау»-моменты)

1. **Hero с анимированным дисплеем кондиционера** — сразу даёт ощущение «живого» и современного сайта.
2. **Квиз-подборщик за 60 секунд** (#quiz) — 6 шагов, прогресс-бар, авто-переход на следующий шаг при выборе. Это та самая механика, что обещали в письме.
3. **Калькулятор стоимости в реальном времени** (#calc) — двигаешь ползунки, цифры обновляются мгновенно. Учитывает площадь, потолки, солнечность, класс монтажа и техники.
4. **Три уровня монтажа** (#montage) — Экономный / По ТУ / По ГОСТ. Заказчик сразу видит, за что платит.
5. **Таймлайн двухэтапного монтажа** — визуально объясняет сложную услугу за секунды (на сайте Altera это стена текста).
6. **Фильтры каталога** — Все / До 40к / Средний / Инверторные / Премиум.
7. **Реальные отзывы с Яндекс.Карт** — сохранили имена и даты.
8. **Плавающие кнопки WhatsApp / Telegram / Звонок** — конверсия с мобилок.
9. **Адаптивный дизайн** — открой на телефоне, всё работает.
10. **Микроанимации появления** при скролле, hover-эффекты, маска телефона в формах.

## Как открыть локально

Просто открой `index.html` двойным кликом. Всё работает без сервера.

Или из терминала:
```cmd
cd altera-redesign
start index.html
```

## Как залить в интернет за 2 минуты (бесплатно)

### Вариант 1 — Netlify Drop (самый быстрый)
1. Открой https://app.netlify.com/drop
2. Перетащи папку `altera-redesign` в окно
3. Получишь ссылку вида `https://random-name.netlify.app` — скидывай клиенту

### Вариант 2 — Vercel
1. Установи `npm i -g vercel`
2. Из папки проекта: `vercel`
3. Логинься, подтверждай — получаешь ссылку

### Вариант 3 — GitHub Pages
1. Создай репозиторий, залей файлы
2. Settings → Pages → Source: main branch
3. Через минуту работает на `username.github.io/repo`
